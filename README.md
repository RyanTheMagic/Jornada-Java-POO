# ☕ Jornada-Java-POO

> Repositório responsável por documentar minha jornada de estudos na linguagem Java

---

### 📝 Sobre o Projeto

A ideia deste repositório veio depois de um grande hiperfoco nos estudos de Java seguido de estagnação causada pela procrastinação. Com este repositório meu objetivo agora é documentar cada passo que eu dou no meu aprendizado em Java, vale ressaltar que estou estudando pelo Curso do Professor Nélio Alves na plataforma Udemy, quando estudei o curso pela primeira vez cheguei bem longe (Parei na aula 135 da Seção 12 - Enumerações, Composições), porém, por uma serie de fatores, deixei o curso de escanteio. Dessa forma, refleti que seria viável a ideia de começar do zero para não deixar nada que eu aprendi anteriormente passar batido.

---

### 📂 Organização do Repositório

Este repositório está dividido em 3 pastas:

* **`EmCurso`** - Essa pasta armazena todo e qualquer "projeto" java que vou realizar ao decorrer de uma aula, ou seja, testar na prática o que está sendo ensinado. Além de ter um comentário no início de cada código indicando qual aula e seção aquele conteúdo foi passado.
* **`Exercicios`** - Essa pasta armazena todos os exercícios do curso, seja exercícios de exemplo ou exercícios práticos das seções (Essa pasta não armazenará exercícios manuscritos).
* **`Meus Projetos`** - Autoexplicativo, essa pasta armazenará qualquer ideia que eu tenha e queira transformar em projeto no Java.
